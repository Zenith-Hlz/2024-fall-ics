A.
    n = 0.yyyyy...

    n << k = y.yyyyy... = Y + n

    (n << k) - n = Y

    两边同除以2^k: n - n/(2^k) = Y/(2^k)

    n(1 - 1/2^k) = Y/(2^k)

    n = Y/(2^k) * (2^k)/(2^k - 1)

    n = Y/(2^k - 1)


B.
    (a) Y = 5, k = 3
	n = 5/(2^3 - 1) = 5/7
	
    (b) Y = 6, k = 4
	n = 6/(2^4 - 1) = 6/15 = 2/5
		
   （c) Y = 19, k = 6
	n = 19/(2^6 - 1) = 19/31
